# rust-ffi-examples

FFI examples written in Rust

|C type                 | Corresponding std::os::raw type |
|-----------------------|--------------------------------:|
|short                  | c_short                         |
|int                    | c_int                           |
|long                   | c_long                          |
|unsigned, unsigned int | c_unit                          |
|void *, const void *   | *mut c_void, *const c_void      |
